# Chrollo

**Chrollo** is a Postman-like tool for inspecting, testing, and debugging **WebSocket APIs**, built specifically for the **SCOPE Platform**. It supports both **web** and **desktop** environments using **React** and **Electron.js**.

- [Electron](https://www.electronjs.org/)
- [React](https://reactjs.org/)
- [WebSocket Protocol (MDN)](https://developer.mozilla.org/en-US/docs/Web/API/WebSockets_API)
